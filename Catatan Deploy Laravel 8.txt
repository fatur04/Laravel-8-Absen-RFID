Cara deploy laravel 8 
folder ada di C:\xampp\htdocs\Laravel-8-Absen
Server Requirements : PHP >= 7.3


Setting mysql
1. jalankan xampp
2. buka browser http://localhost/phpmyadmin/
3. buat database dan import database di folder C:\xampp\htdocs\Laravel-8-Absen\coba.sql


buka vs code. pilih folder C:\xampp\htdocs\Laravel-8-Absen
1. copy file env.example
2. rename file menjadi .env
3. setting database, ganti nama database sesuai nama pada mysql
4. save

Run cmd/terminal pada folder C:\xampp\htdocs\Laravel-8-Absen
command line:
1. composer install --optimize-autoloader --no-dev
2. php artisan config:cache
3. php artisan route:cache
4. php artisan view:cache
5. php artisan serve --port=8001
6. run di browser ip http://127.0.0.1:8001
7. php artisan optimize

Sharing Aplikasi 
1. php artisan serve --host=10.11.7.3 --port=8001
2. run di browser ip http://10.11.7.3:8001


Bersihkan route cache
1. php artisan cache:clear
2. php artisan route:cache
3. php artisan config:cache
4. php artisan view:clear
